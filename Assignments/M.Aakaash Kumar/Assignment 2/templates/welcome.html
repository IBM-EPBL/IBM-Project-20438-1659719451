How to connect db2 with python

conn=ibm_db.connect("DATABASE=*****;HOSTNAME=*****;PORT=88888;SECURITY=SSL;SSLServiceCertificate=DigiCertGlobalRootCA.crt;UID=*******;PWD=********",'','')
BY using above line we can connect to db2 using python.
The details are found in ibm.cloud.com->resources->databases->service credentials.

Rendering html pages using python

For rendering html pages in python, we need to import render_templates from flask
Example:render_templates('welcome.html')

Redirecting to another webpage

For redirecting to another webpage, we need to import url_for and redirect from flask
Like "from flask import url_for,redirect"
Example:redirect(url_for("admin"))
